范票规则

  • 如何使用

  • 如何赚取

1、购物抵扣

范票可直接用于支付团洋范商品订单，范票和现金抵扣的比例为100:1，可使用范票数量为100的整数倍，如100、200、300等范票数，超出1000则不受此限制。

2、如何使用

在订单结算页面“使用范票/抵用”处，选择范票数量，即可减免相应付款金额，范票数量小于100是看不到的哦 ~ 具体如下图：

1、购买商品

通过分享链接购买的，当购买商品后用户和店铺可享受范票回馈，范票回馈值为10 (100张范票等于1元现金)，店长或掌柜在自身店铺购买则只享受单向范票回馈 (10)，取消订单则会扣除相应范票 。

2、商品评价

使用商品评价功能，我们将在您评价成功后对店铺和您给予相应的范票回馈，范票反馈值为5，用户只能评价该用户名下购买过的商品 。

3、平台游戏

通过团洋范开放的游戏可获得范票回馈，回馈值可参考相应的游戏规则。

4、有奖征集

关注团洋范或团洋范微讯公众号，参加正在进行的「有奖征集活动」，可获得与活动规则规定的范票回馈。

5、红包互动

通过平台的红包功能，可与朋友圈或微信群的朋友进行红包互动，拆开红包后可获得相应的范票收入。

提示：小范票，大有用，多赚一些囤起来~~
